# Agent Protocols & Documentation

The `.agent/` directory is an advanced developer space used to dictate, store, and distribute operational metadata for Autonomous AI Agents contributing to MarsinDictation. 

It is kept public to allow end-to-end transparency in how the project design choices, privacy implementations, and deployment automations are contextually parsed by Large Language Models interacting with the codebase.

## Directory Structure
- **`00_gol/`**
  Contains the core "Game of Life" parameters dictating our privacy constraints, telemetry rules, and git contribution procedures. Agents must strictly obey these references to ensure the Open Source standards of the `main` branch are protected.
- **`01_designs/`**
  Technical breakdowns of planned logic mechanisms (like silence ducking and embedded Whisper models).
- **`03_reports/`**
  Historical action logs summarizing completed workloads, performance checks, and gap analyses spanning Windows and macOS implementations.
